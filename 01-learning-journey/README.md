# Learning Journey

Documentation of the BLKOUT platform development journey.

## Structure

- `deployment/` - Deployment attempts and configurations (versioned)
- `extensions/` - Browser extension development history
- `campaigns/` - Past campaign materials and learnings

## Versioning Convention

Documents are numbered to show evolution:
- `v1-` Initial attempts
- `v2-` First iterations
- `v3-` Refinements
- `v4-` Production versions
