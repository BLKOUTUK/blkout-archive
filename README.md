# BLKOUT Platform Archive

This repository preserves the learning journey and development history of the BLKOUT platform.

---

## Purpose

- **Historical Reference**: Document iterations and evolution of the platform
- **Learning Journey**: Preserve development decisions and their context
- **Version History**: Maintain numbered versions of key documents
- **External Storage**: Reduce main repository size by moving non-active files

---

## Structure

```
blkout-archive/
├── 01-learning-journey/
│   ├── deployment/           # Deployment attempts and guides (versioned)
│   ├── extensions/           # Old extension versions
│   └── campaigns/            # Past campaign materials
├── 02-brand-assets/
│   ├── images/               # Archived brand images
│   └── videos/               # Campaign videos
└── 03-deprecated/
    ├── old-configs/          # Deprecated configurations
    └── unused-features/      # Features not taken forward
```

---

## Document Versioning

Documents are numbered to show their evolution:
- `v1-initial-attempts.md`
- `v2-vercel-setup.md`
- `v3-coolify-migration.md`
- `v4-production-config.md`

---

## Restoring Files

If you need to restore archived files to the main platform:

1. Locate the file in the appropriate archive folder
2. Copy back to the original location in `blkout-platform`
3. Update any imports or references as needed
